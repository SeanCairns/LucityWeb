Implementation Specialist

We are accepting resumes for the position of Implementation Specialist. This position should establish a positive baseline for the relationship between the client and Lucity all while supporting the project plan and schedule for implementing the Lucity software.  Responsibilities would include:

  • Provide technical and project management services to ensure successful and profitable implementation of Lucity software products for our clients.
  • Scheduling/coordinating project efforts and activities with the Lucity implementation project stakeholders client-side
  • Product support for entire Lucity product line (includes new products/functionality/platforms that we add at each release)
  • All project implementation activities including (but not limited to):
  • Training end users and administrators on-site and via the web
  • Conversion of existing client data
  • Business process review
  • Coordination/Management of all project activities and schedule/timeline
  • Create project documentation when required

If you are a technical person who enjoys (and is good at) working with people, keeping up with technology, managing projects and traveling, this job is for you!

Desired Skills and Experience:

  • Hands-on experience providing technical services for implementation of COTS products
  • Hands-on experience providing management services for implementation projects, preferably to local governmental agencies, is preferred (but not required)
  • Exceptional presentation and training skills
  • Excellent interpersonal and communication skills
  • Strong problem solving and analytical skills
  • Ability to travel extensively
  • Proficiency with Microsoft Office (Word, PowerPoint, Excel, Access)
  • Experience with databases (specifically Microsoft Access, SQL Server and Oracle)
  • Experience with Crystal Reports
  • Experience with Esri software is required (ArcGIS and ArcServer at a minimum)

Required Travel: 50-75%

Required Education: BS/BA degree with GIS emphasis
